环境配置

>

  • 首先在本机安装nodejs及npm

  • 安装完成后运行

    npm install -g bakari-builder

    >

  • 从svn下载最新项目代码 svn地址 : http://svn.mizhe.com/repos/mizhe/trunk
  • 进入项目根目录如：cd D://mizhe

  • 执行bakari命令

    //此命令用来生成测试环境代码及线上压缩代码
bakari build
    具体bakari的使用方法请参考 https://github.com/EarlyH/bakari-builder
